# AUREN — Premium Minimalist E-Commerce Demo

AUREN is a complete multi-page e-commerce storefront and admin dashboard built using only **plain HTML, CSS, and JavaScript**, with **Chart.js** used only for admin analytics charts.

The project is designed as a polished portfolio-ready front-end demo for a premium minimal brand selling sneakers, apparel, bags, and accessories.

## Live Demo

- **GitHub Pages:** https://shubham-dev00.github.io/premium-ecommerce-website/

## Live Preview Note

This is a static demo project. To preview locally, just open `index.html` in your browser or run it with **VS Code Live Server**. For the smoothest experience, keep your internet connection on so Google Fonts and Chart.js CDN can load.

## Features

- Modern Apple-like premium UI
- Fully responsive customer storefront and admin dashboard
- Shared design system with CSS custom properties
- Sticky header, mobile menu, custom scrollbar, back-to-top button
- Glassmorphism cards, overlays, modals, and panels
- Page transitions, ripple effects, scroll reveal, count-up animation
- Product-card mouse-tracking 3D tilt interaction
- Skeleton loading states on product grids
- Live navbar search autocomplete with local search history
- Product listing filters, sorting, pagination, active tags, quick view
- Product detail gallery, zoom effect, tabs, reviews, related items, recent views
- Cart system with localStorage persistence
- Wishlist system with localStorage persistence
- Dummy auth system with signup, login, logout, route protection
- Checkout flow with 3 steps and validation
- Order confirmation and order tracking timeline
- Account page, address management, order history actions
- FAQ search, blog filtering, contact form validation, newsletter validation
- Admin dashboard with metrics, recent orders, low-stock alerts, quick actions
- Admin products, orders, customers, coupons, analytics, and settings pages


## Pages Included

### Customer Pages
- `index.html` — premium homepage with hero, flash sale timer, categories, best sellers, new arrivals, testimonials, logos, newsletter
- `products.html` — shop page with filters, sorting, pagination, quick view
- `product-detail.html` — gallery, zoom, tabs, size guide, pincode checker, related products
- `search-results.html` — search results layout using the same product engine
- `cart.html` — cart, saved items, promo code, totals, recommendations
- `checkout.html` — 3-step checkout flow with validation
- `order-confirmation.html` — animated success state with order summary
- `order-tracking.html` — order tracking timeline by order ID
- `wishlist.html` — saved products page
- `login.html` — demo login UI with password toggle and social buttons
- `signup.html` — signup form with validation and password strength bar
- `forgot-password.html` — reset-link demo form
- `account.html` — customer profile and address manager
- `order-history.html` — previous orders with actions and expandable details
- `about.html` — brand story, values, team, stats
- `contact.html` — contact form and store info
- `faq.html` — searchable FAQ accordion
- `blog.html` — featured post and category-filtered article cards
- `terms.html` — terms page
- `privacy.html` — privacy page
- `refund-policy.html` — returns and refund page

### Admin Pages
- `admin/dashboard.html` — metrics, revenue chart, order chart, low stock, top products
- `admin/products.html` — product CRUD demo with modal form and bulk delete
- `admin/orders.html` — status updates, filters, order detail modal
- `admin/customers.html` — customer search, profile modal, ban/activate toggle
- `admin/coupons.html` — coupon create/toggle/delete demo
- `admin/analytics.html` — revenue, category, traffic, funnel, top product analytics
- `admin/settings.html` — general, shipping, tax, notifications, payment toggles

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Chart.js (admin charts only)
- localStorage for demo persistence
- Picsum Photos for placeholder product and content imagery
