# AUREN — Premium Minimal E-Commerce Demo

AUREN is a modern, production-style e-commerce website demo built using **plain HTML, CSS, and JavaScript** with no frameworks. It features a premium Apple-inspired design, responsive layouts, glassmorphism UI, micro-interactions, localStorage-powered cart/auth/wishlist systems, and a complete admin panel with analytics charts.

**Tagline:** *Move Light. Live Sharp.*

---

## Live Preview

This project is a static front-end demo.

To preview locally, simply open `index.html` in your browser or run the project using **VS Code Live Server** for the best experience.

> Note: Admin charts use **Chart.js via CDN**, so internet is required for charts to display correctly.

---

## Features

### Customer Features

- Full responsive e-commerce storefront
- Modern premium UI with Apple-like minimal aesthetic
- Sticky navbar with cart and wishlist badges
- Hero section with animated headline and flash sale countdown
- Featured categories, best sellers, new arrivals
- Product listing page with:
  - live search
  - category filters
  - price filters
  - brand, rating, color, and size filters
  - sorting options
  - pagination
- Product detail page with:
  - image gallery
  - thumbnail switching
  - zoom/hover effects
  - size and color selectors
  - quantity stepper
  - pincode delivery estimate
  - tabs for description/specs/reviews/shipping
  - related and recently viewed products
- Cart system with:
  - add/remove/update quantity
  - save for later
  - coupon support
  - GST calculation
  - localStorage persistence
- Wishlist system with localStorage persistence
- Checkout flow with:
  - address validation
  - payment method selection
  - review step
  - order placement
- Order confirmation page
- Order tracking page with timeline
- Login, signup, forgot password pages
- User account page with saved addresses and profile settings
- Order history page
- About, contact, FAQ, blog, and policy pages
- Toast notification system
- Cookie consent banner
- Back to top button
- Skeleton loading states
- Scroll-triggered reveal animations
- 3D tilt product card interactions
- Mobile hamburger navigation

### Admin Features

- Admin dashboard with statistics cards
- Revenue and order charts using Chart.js
- Product management page
- Order management page
- Customer management page
- Coupon management page
- Analytics/reporting page
- Settings page for site configuration
- Local demo data for orders, products, customers, and analytics
