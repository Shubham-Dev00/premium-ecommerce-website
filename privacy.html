Privacy Policy

AUREN stores selected demo information such as cart contents, wishlist items, address entries, search history, and login state inside your browser localStorage to simulate a realistic customer journey. No real server calls are made for account authentication, order creation, or analytics reporting. The storefront does not intentionally collect personal data beyond what you type into forms during the demo. Because this is a static front-end project, clearing browser storage resets most customer-side data. If this demo is adapted for real-world use, production security, backend validation, cookies governance, consent handling, and legal compliance must be implemented before launch.

Use of the demo

This storefront exists as a polished front-end showcase and learning project. Any production deployment should add real authentication, secure checkout processing, backend order validation, rate limiting, observability, and legal review before use with actual customers.

Operational note

All product images are placeholder assets served from Picsum. Brand stories, customer records, order IDs, metrics, and policy copy are synthetic and intended only to make the interface feel complete and realistic.
